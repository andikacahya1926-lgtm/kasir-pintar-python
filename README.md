# Kasir Pintar CLI (Python)

Program kasir berbasis terminal (CLI) sederhana yang dirancang untuk membantu pemilik toko menghitung total belanjaan pelanggan beserta diskon bertingkat secara otomatis.

## 🚀 Fitur Utama
- **Diskon Bertingkat Otomatis:** Potongan 20% untuk belanja ≥ Rp 500.000 dan 10% untuk belanja ≥ Rp 100.000.
- **Perulangan Transaksi (`while loop`):** Kasir dapat melayani banyak pelanggan berturut-turut tanpa harus merestart program.
- **Case-Insensitive Handling (`.lower()`):** Program tetap mengenali input konfirmasi selesai meskipun diketik dengan huruf besar/kecil (misal: "Tidak", "tidak", "TIDAK").

## 🛠️ Teknologi yang Digunakan
- **Bahasa Pemrograman:** Python 3
- **Konsep Dasar:** Variabel, Tipe Data (`int`, `str`), Percabangan (`if-elif-else`), Perulangan (`while`), dan Manipulasi String.
